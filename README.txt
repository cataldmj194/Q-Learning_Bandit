Q_learning.java is my implementation of the Q-learning algorithm for this
assignment.

Usage:
for compilation, issue the following command
javac Q_learning.java

run the program in the following format:
java Q_learning <NameOfTextFile>

I have provided several input files for testing purposes. Those files are:
data.txt (this one is fun, made a room out of obstacles)
data2.txt
compound.txt
